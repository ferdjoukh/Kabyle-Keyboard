Qu'est ce que c'est
===================

Ceci est un clavier kabyle pour Linux. Il permet d'écrire l'alphabet Kabyle Latin (Tamεamrit)


Fonctionnement
==============

- ẓ = alt gr + z
- ṛ = alt gr + r
- ṣ = alt gr + s
- ḥ = alt gr + h

- ǧ = alt gr + g
- č = alt gr + c

- γ = alt gr + q
- ε = alt gr + w
